## What is a LinkedIn Link Preview?

A LinkedIn link preview is the visual card that appears when you share a URL in your post. It displays:

- **Featured image** - the main visual (pulled from OpenGraph tags)
- **Title** - the headline of the linked content
- **Description** - a short summary of what the link is about
- **Domain** - the website URL

This preview card appears directly in the LinkedIn feed, making your shared content more clickable and visually appealing than a bare URL.

## Why LinkedIn Link Previews Matter for Engagement

Link previews dramatically impact your post's performance:

- **Posts with link previews get 45% more clicks** than posts with plain URLs
- **Visual content attracts attention** - users scroll fast, images stop them
- **Trust signal** - a professional-looking preview shows you're sharing quality content
- **Context** - the preview tells users what they'll get before they click

A broken or poorly optimized link preview sends the wrong message. It looks unprofessional, reduces clicks, and wastes the effort you put into creating the content.

## How LinkedIn Generates Link Previews

When you paste a URL into LinkedIn, the platform scrapes the webpage for **OpenGraph meta tags**. These are special HTML tags that define how links should appear when shared on social media.

LinkedIn looks for these specific tags:

- `og:title` - the title text
- `og:description` - the description text
- `og:image` - the image URL
- `og:url` - the canonical URL

If your webpage has these tags properly configured, LinkedIn pulls the data and generates a clean preview. If not, LinkedIn tries to guess by scraping the page title, first image, and meta description - often with messy results.

## How to Optimize Your LinkedIn Link Preview

If you control the webpage you're linking to (your blog, product page, etc.), optimize the OpenGraph tags for better previews.

### Set Proper OpenGraph Tags

Add these meta tags to your page's `<head>` section:

```html
<meta property="og:title" content="Your Compelling Title Here" />
<meta property="og:description" content="A clear, benefit-focused description under 200 characters" />
<meta property="og:image" content="https://yoursite.com/image.png" />
<meta property="og:url" content="https://yoursite.com/page" />
```

### Optimize Your Preview Image

LinkedIn recommends:

- **Dimensions:** 1200 x 630 pixels (1.91:1 aspect ratio)
- **Format:** PNG or JPG
- **File size:** Under 5 MB
- **Content:** Clear, high-contrast, readable text if any

Avoid tiny images, vertical images, or images with important details at the edges (LinkedIn crops them).

### Write Click-Worthy Titles and Descriptions

- **Title:** Keep it under 60 characters so it doesn't get cut off
- **Description:** 150-200 characters, explain the benefit clearly
- **Use action words** - "Learn how to...", "Get the guide...", "Discover why..."

### Common Mistakes to Avoid

- Using generic placeholder images
- Titles that don't match the actual content
- Missing OpenGraph tags entirely
- Image URLs that return 404 errors
- Descriptions that are too vague or too long

## How to Preview Your LinkedIn Link Before Posting

## Troubleshooting LinkedIn Link Preview Issues

### Preview Not Showing

If LinkedIn doesn't display a preview card:

- The URL might be blocked or return an error
- The webpage might have `noindex` or `nofollow` tags
- LinkedIn's scraper couldn't access the page (check robots.txt)

Try posting the link in a test post (visible only to you) to see if it generates a preview.

### Wrong Image or Title

If LinkedIn shows the wrong image or an old title:

- **LinkedIn caches link previews** - even after you update your OpenGraph tags, LinkedIn might show the old version
- **Clear the cache:** Use [LinkedIn's Post Inspector](https://www.linkedin.com/post-inspector/) to force a refresh
- Paste your URL, click "Inspect", and LinkedIn will re-scrape the page

### Missing Description

If the description is blank:

- Add or fix the `og:description` meta tag
- Make sure it's in the `<head>` section, not the `<body>`
- Check for HTML syntax errors that might break the meta tag

## FAQ: LinkedIn Link Previews

### How do I change my LinkedIn link preview?

To change your LinkedIn link preview, you need to update the OpenGraph meta tags (`og:title`, `og:description`, `og:image`) on the webpage you're linking to. After updating, use LinkedIn's Post Inspector to refresh the cache.

### Why is my LinkedIn link preview not showing?

Common causes: the webpage blocks LinkedIn's scraper (check robots.txt), the page returns an error, or the OpenGraph tags are missing or broken. Test the URL in a LinkedIn preview tool to diagnose the issue.

### Can I preview LinkedIn links without posting?

Yes. Use a LinkedIn link preview tool like [linkedinpreview.com](/#tool) to paste your URL and see exactly how the preview will appear - without creating a live post.

### What size should LinkedIn preview images be?

LinkedIn recommends 1200 x 630 pixels (1.91:1 aspect ratio) for optimal display on desktop and mobile. Minimum size is 200 x 200 pixels, but smaller images look pixelated.
